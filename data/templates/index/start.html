Getting started

Installation script

curl -fsSL https://vinceanalytics.com/install.sh | bash

Homebrew

brew install vinceanalytics/tap/vince

Container image

docker pull ghcr.io/vinceanalytics/vince

Usage

Usage of vince:
  -acme
        Enables auto tls. When used make sure -acme.email and -acme.domain are set
  -acme.domain string
        Domain name to use with lets encrypt
  -acme.email string
        Email address to use with lets enctrypt
  -admin.bootstrap
        Creates admin account on startup
  -admin.email string
        Email address for admin account
  -admin.name string
        User name for admin account
  -admin.only
        Disables registration
  -admin.password string
        Password for admin account
  -data string
        Path to where database data is stored (default ".data")
  -db.gc duration
        Interval for running GC checks on value log (default 1m0s)
  -listen string
        tcp address to bind the server (default ":8080")
  -sys.interval duration
        Interval for collecting system stats (default 15m0s)

Starting server

vince
